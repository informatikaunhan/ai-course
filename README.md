## Deskripsi

Mata kuliah ini memperkenalkan prinsip-prinsip dasar, teknik, dan aplikasi kecerdasan artifisial. Mahasiswa mempelajari konsep agen cerdas, algoritma pencarian, representasi pengetahuan, penalaran probabilistik, serta dasar-dasar pembelajaran mesin. Materi disajikan dengan konteks aplikasi pertahanan dan militer.

**Prasyarat:** Struktur Data dan Algoritma (SDA201), Statistika dan Probabilitas (STP301)
